Help & Instructions

Using the Information Cards System

  • Add cards: Click the + button or press Ctrl+N
  • Search: Type in the search box and press Enter
  • Filter: Use the category dropdown to filter cards
  • Pin cards: Click the thumbtack icon to pin important cards
  • Export: Click the export button to save all cards as JSON
